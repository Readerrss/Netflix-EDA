# Netflix-EDA
Interactive Netflix EDA dashboard analyzing movies and TV shows (Nov 2019). Explore dataset previews, cleaning steps, visualizations for genres, countries, ratings, durations, top actors/directors, correlations, and global distribution using Python, Seaborn, Plotly, and Streamlit.
